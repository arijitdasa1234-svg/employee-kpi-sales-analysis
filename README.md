# 📊 Employee KPI & Sales Performance Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an employee performance dataset containing information about **5,000 employees**. Using Python and popular data analysis libraries, the project uncovers insights into employee performance, salaries, attendance, bonuses, KPIs, and sales achievements.

The goal is to demonstrate practical data analysis skills by cleaning data, performing statistical analysis, creating visualizations, and generating actionable business insights.

---

## 🎯 Objectives

* Analyze employee performance metrics.
* Identify top-performing and underperforming employees.
* Compare Sales Target vs. Sales Achieved.
* Categorize employees based on Monthly KPI.
* Explore relationships among salary, performance, and sales.
* Generate business insights using data visualization.

## 📌 KPI Categories

Employees are categorized based on their Monthly KPI score.

| KPI Score | Category             |
| --------- | -------------------- |
| 90 – 100  | ⭐ Best               |
| 80 – 89   | ✅ Good               |
| 70 – 79   | 🟡 Average           |
| 60 – 69   | ⚠️ Needs Improvement |
| Below 60  | ❌ Poor               |

---

## 📈 Key Insights

* Most employees maintain high attendance.
* Salary distribution is close to a normal distribution.
* Higher KPI scores generally correspond to better performance.
* Several employees exceeded their sales targets.
* KPI categorization helps identify high performers and employees requiring improvement.
